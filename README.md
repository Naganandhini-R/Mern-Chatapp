# Real-Time Chat Application

A full-stack real-time chat application built with the MERN stack (MongoDB, Express, React, Node.js) and Socket.IO for instant messaging and seamless communication.

## Features
- User signup & login authentication
- Real-time one-to-one messaging
- Online users status
- Secure JWT authentication
- Protected routes & API security
- Responsive modern chat UI
- Persistent login using cookies

## Tech Stack

### Frontend
- React + Vite
- Tailwind CSS

### Backend
- Node.js + Express

### Database
- MongoDB with Mongoose

### Authentication
- JWT Authentication

### Real-Time Communication
- Socket.IO

## Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
